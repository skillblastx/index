# Skill Blast X - Página oficial del juego 
Página principal del juego "Skill Blast X"

¿Qué contiene?
1. *"Home" con descripción de la mecánica de juego, cartas, personajes y contacto.*
2. *Sección "cartas" que despliega todas las cartas disponibles a la fecha del juego.*
3. *Sección "torneos" que describe todos los eventos realizados en Exodia House, junto con los ganadores y fotografías de la comunidad.*
4. *Sección "banlist" que muestra las cartas prohibidas y limitadas del juego.*

Lenguajes utilizados:
1. HTML
2. CSS
3. Javascript
4. Motion One
